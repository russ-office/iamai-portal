The atom of Pulse list & overview pages: a 40px row with an optional status dot, primary/secondary text, mono meta, and a trailing slot. Stack rows directly (each carries its own bottom hairline).

```jsx
<DataRow status="ok" primary="Dana Okoro" secondary="Payments" meta="4 open · 6d" />
<DataRow status="bad" primary="Auth rewrite" meta="PR-4821" onClick={open} />
```
